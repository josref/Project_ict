Les 1

Tijdens onze eerste les van het project ICT kregen we uitleg over HTML en CSS. We leerden hoe we zelf een simpele website kunnen programmeren met HTML. De website bevatte links waar je op kon drukken om naar andere delen van de website te gaan. Ook voegden we een foto toe. Verder zagen we hoe je de opmaak van een website makkelijk kunt aanpassen met CSS. Zo kun je eenvoudig de lettergrootte, kleur van de letters en achtergrond aanpassen.
