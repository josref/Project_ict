Les 2

In de tweede en laatste les ICT maakten we kennis met PHP, dit kan helpen om dingen op een website te automatiseren. Ook konden we onze website nu hosten op onze eigen computer. Zo konden we zien hoe onze website er uit zou zien moesten mensen hem bezoeken op het internet. Tijdens de les programmeerden we ook een formulier op een website voor een afhaalrestaurant. Hierop konden mensen aanduiden wat ze wouden eten en wanneer ze het wouden ophalen.
