Les 4

Tijdens deze les leerden we hoe we onze arduino kunnen programmeren. We leerden hoe we simpele codes moeten schrijven om bijvoorbeeld een led te laten knipperen. Ook zagen we dat je de waarden van de verschillende censoren kunt doorsturen via een antenne. Hierdoor kun je bijvoorbeeld iemand op afstand laten weten hoe warm het is en wat de lichtintensiteit op de locatie van de arduino is. We maakten ook een programma om gordijnen automatisch dicht te doen als het donker wordt.
